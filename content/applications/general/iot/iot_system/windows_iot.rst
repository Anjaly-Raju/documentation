
===========
Windows IoT
===========

Windows IoT is a Windows program installable on a Windows machine which allow to use your Windows computer as an IoT system.

Support Scope
=============

As the computer is not an isolated system (as opposed to the IoT box), other installed software can interfere with the IoT system.
We will assume that no third-party software is installed on the Windows IoT device and the default Windows settings/firewall are used.
